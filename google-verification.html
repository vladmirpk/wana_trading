Google Search Console Verification

This page is used for Google Search Console verification.

Replace "YOUR_VERIFICATION_CODE_HERE" with your actual verification code from Google Search Console.

How to get your verification code:

  1. Go to Google Search Console
  2. Add your property: https://wanatrading.co.zw
  3. Choose "HTML tag" verification method
  4. Copy the verification code
  5. Replace "YOUR_VERIFICATION_CODE_HERE" in this file
  6. Upload the file to your website
  7. Click "Verify" in Google Search Console

After verification:

  • Submit your sitemap.xml
  • Monitor search performance
  • Fix any crawl errors
  • Track your rankings
